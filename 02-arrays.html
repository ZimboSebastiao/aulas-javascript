Arrays (vetores e matrizes)

Conceito

Vetor (ou Array) é uma estrutura de dados que permite armazenar em memória uma lista de informações. Pode ser declarado como variável e/ou constante, e cada um dos dados/informações armazenados possui uma referência chamada de índice.

Array de uma única dimensão é chamado de vetor, enquanto que com duas ou mais dimensões, é chamado de matriz.
